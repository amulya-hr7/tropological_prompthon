# Topological: Semantic News Navigator

## AI-Powered Semantic Search & News Visualization System

🔗 Live Demo: https://interactive-search-n-mi9m.bolt.host

---

## 🚀 Overview

This project is an AI-powered semantic search engine that understands the meaning of news articles instead of keywords.

It visualizes relationships using embeddings + PCA clustering.

---

## 🎯 Features

- Semantic search (meaning-based retrieval)
- AI embeddings using sentence-transformers
- Vector database (ChromaDB)
- Topic clustering using ML
- 2D PCA visualization
- Interactive news map

---

## ⚙️ Tech Stack

Backend: FastAPI, Python  
AI: Sentence Transformers  
Database: ChromaDB  
ML: Scikit-learn (PCA)  
Frontend: React (Vite)  
Visualization: Plotly.js

---

## 🧠 How It Works

User Query → Embedding → Vector DB → Similarity Search → PCA → Visualization
